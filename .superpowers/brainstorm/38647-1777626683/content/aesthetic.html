Visual aesthetic for the pre-launch carousel

The existing carousel is dark (#1c1c1e), red accent (#c0392b), geometric grid lines, monospaced caps

A

Match exactly

Same dark background, red accent, corner brackets, grid lines. Feels cohesive — like it belongs to the same brand moment. Familiarity builds recognition.

B

Darker / more stripped back

Pure black background, red reduced to a single glow or line, fewer elements. More ominous. The "before the lights come on" version of the brand.

C

Red-dominant / high voltage

Flip the palette — red as the primary fill, dark as contrast. Signals a rupture, something new arriving. More aggressive energy for the launch moment.
